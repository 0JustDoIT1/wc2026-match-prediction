# ⚽ 2026 FIFA World Cup Match Prediction

> 머신러닝을 활용한 2026 FIFA 북중미 월드컵 경기 결과 예측 및 토너먼트 시뮬레이션 프로젝트

---

## 📌 프로젝트 개요

1872~2026년 국제 축구 경기 데이터(45,797경기)를 기반으로
2026 FIFA 월드컵 전 경기의 승/무/패 확률을 예측하는 3-Class 분류 모델을 구축하고,
이를 몬테카를로 시뮬레이션으로 확장해 조별리그 진출부터 우승까지의 단계별 확률을 도출합니다.

단일 경기 예측만으로는 "한국이 우승할 확률은 몇 %인가" 같은 다단계 질문에 답할 수 없습니다.
토너먼트는 매 라운드 확률이 누적으로 곱해지는 구조이기 때문에, 분류 정확도보다
**확률 추정 자체의 정교함(Log Loss)** 이 시뮬레이션 신뢰도의 핵심입니다.

---

## 🎯 분석 목표

1. 승/무/패 확률을 예측하는 3-Class 분류 모델 구축
2. 몬테카를로 시뮬레이션으로 조별리그 진출 → 토너먼트 단계별 진출 → 우승까지의 확률 정량화
3. 핵심 산출물로 대한민국 대표팀의 32강 진출 확률 및 우승까지의 경로별 확률 도출
4. 모델 해석(SHAP)을 통한 예측 변수의 영향력 분석
5. 이변(Upset) 요인 분석 및 역대 우승팀 공통 요인 분석 *(팀 내 별도 분석)*

## ⚙️ 주요 피처 (52개)

| 카테고리 | 피처 | 비고 |
|---|---|---|
| Elo 레이팅 | `delta_elo`, `home_elo`, `away_elo` | eloratings.net 스크래핑(234개국) 기반 |
| 단기 폼 | `home_form`, `away_form` (EWMA, span=5) | `shift(1)`로 데이터 누출 방지, 결측 시 중립값(1)+플래그 |
| 직전 4년 성적 | 득점·실점·승무패·득실차 등 18개 | 2026-06-10 기준 직전 4년 |
| WC 경험 | `wc_participations`, `wc_titles` | |
| 경기 환경 | `neutral`, `is_wc`, `is_host` | |
| Gap 피처 | 홈/원정 차이 7개 | |
| 대륙 원핫 | AFC·CAF·CONCACAF·CONMEBOL·OFC·UEFA | 6개 연맹 × 홈/원정 |

**결측치 처리 원칙**
- Elo 결측(3,675행, 약 8%) → **제거** : 팀 실력을 나타내는 핵심 변수라 임의값 채우기는 실력 차이를 왜곡할 위험
- 폼(form) 결측 → **중립값(1.0) + `_form_missing` 플래그** : 역사가 짧은 팀에서 흔히 발생하며, 제거 시 2026 WC 실제 출전팀(신생국 등)의 경기 자체가 데이터에서 소실되는 문제

**Train/Test 분리**
- 시간 기반 분리(`SPLIT_DATE=2022-01-01`), train 41,299 / test 4,498
- 예측 대상이 월드컵(미래 경기)이므로, 과거로 미래를 맞히는 구조를 검증 단계에도 동일하게 적용 — 랜덤 분리는 이 구조를 깨뜨려 데이터 누수를 유발

> (경기 당일 우연성, 양 팀의 신중한 운영 전략 등)이 작용하는 것으로 해석됩니다.

## 🔬 모델별 핵심 설계

**Random Forest / XGBoost / LightGBM**
- 동일한 Optuna 설정(50 trials, Log Loss 최소화, TPESampler+MedianPruner)으로 공정 비교
- 탐색 파라미터 범위는 알고리즘 본질에 맞게 차별화: RF(배깅, 깊은 트리 허용) vs XGB/LGB(부스팅, 얕은 트리)
- LightGBM은 `num_leaves`(리프-와이즈 성장 고유 파라미터)와 Early Stopping을 추가 적용

**Poisson 회귀**
- 홈/원정 득점을 각각 독립 모델(`PoissonRegressor`)로 학습해 기대득점 λ 추정
- `min_matches≥3` 필터링: GLM은 모든 행에 동일 회귀계수가 곱해지는 구조라 표본이 적은 팀의
  극단치에 취약 → 4년간 3경기 미만 매치업 1,004건(2.4%)을 학습에서만 제외(평가는 전체 test set 유지)
- StandardScaler + alpha=1e-4: 피처 스케일 불균형(delta_elo ±1200대 vs neutral 0~1)으로 인한
  과도한 정규화(계수가 0으로 눌리는 현상)를 해결

**Dixon-Coles**
- 기본 Poisson은 홈/원정 득점을 독립으로 가정해 저득점 무승부(0-0, 1-1)를 과소추정
- 0-0/0-1/1-0/1-1 네 칸에만 보정계수 ρ를 적용해 행렬을 재정규화
- `scipy.optimize.minimize`(L-BFGS-B)로 test set Log Loss를 최소화하는 ρ를 탐색 (범위 −0.5~0)
- 최적값 ρ=−0.1472, 평균 Draw 확률 0.212 → 0.241로 개선 (실제 0.229에 근접)

## 🧠 모델 해석 (SHAP / Feature Importance)

- `delta_elo`가 XGBoost·LightGBM 모두에서 전체 피처 중 압도적 1위
- 모델별 2위 피처는 다르게 나타남: XGBoost는 `neutral`(중립 경기장 효과), LightGBM은 `gap_form`
  (단기 폼 차이) — 동일 데이터에도 두 부스팅 알고리즘의 트리 성장 방식 차이(레벨-와이즈 vs
  리프-와이즈)가 반영된 것으로 해석
- Draw 클래스는 SHAP value 분산이 좁고 약하게 나타나, Draw F1이 구조적으로 낮은 원인과 연결

## 🎲 몬테카를로 시뮬레이션

**STEP 1 — 조별리그 (N=10,000)**
- 최종 앙상블 확률로 승/무/패를 결정하고, Dixon-Coles 스코어 행렬에서 그 결과에 해당하는
  스코어만 조건부 샘플링(`sample_score`)해 득실차까지 정밀 반영
- 승점 → 득실차 → 다득점 → 무작위 순으로 순위 결정(실제 FIFA 규정과 동일)
- 3위 8팀은 12개 조 3위 팀을 매 회차 다시 비교해 상위 8팀만 진출

**STEP 2 — 토너먼트 32강~결승 (N=10,000)**
- 대진 **구조**(매치업 규칙)는 고정이지만, 그 안에 들어가는 **참가팀**은 매 회차 조별리그부터
  전체를 다시 시뮬레이션 — 고정된 단일 대진표가 아닌, 조별리그+토너먼트 전체의 불확실성을 반영
- 3위 진출팀의 슬롯 배정은 제약 조건(특정 조 출신만 가능) 충돌을 해결하는 백트래킹 알고리즘 사용
- 토너먼트는 무승부가 없어 무승부 확률을 홈/원정 승리 확률에 비례 재분배(승부차기 근사)
- 총 소요시간 약 119분(10,000회)

**핵심 결과 — 대한민국**

| 단계 | 확률 |
|---|---|
| 32강 진출 (조 1·2위 34.8% + 3위 진출 45.8%) | **80.5%** |
| 16강 진출 | 22.04% |
| 8강 진출 | 3.66% |
| 4강 진출 | 0.26% |
| 우승 | 0.00%* |

\* 반올림 표기이며 실제로는 0.005% 미만 — 10,000회 시뮬레이션 중 등장 빈도에 기반한 한계로,
진짜 확률이 정확히 0이라는 의미는 아닙니다.

**실시간 업데이트**: 실제 A조 1·2차전 결과(한국 2-1 체코, 멕시코 1-0 한국 등)를 반영해 3차전만
재시뮬레이션한 결과, 한국의 조 1·2위 확률이 34.8% → **89.0%**로 상승. 다른 조는 진행 단계가
달라(대부분 1경기만 진행) 전체 재계산은 보류했습니다(시도 시 97.1%로 과대평가됨을 확인).

**전체 우승확률 상위 4개국**: 스페인(45.99%), 프랑스(16.43%), 잉글랜드(10.30%), 아르헨티나(9.87%)

**3위 제도의 구조적 효과**: 48팀·3위 8팀 제도는 조 추첨이 불리했던 팀에게도 만회 기회를 제공
— 모로코(조1·2위 확률 34.0% → 최종 진출 75.3%), 코트디부아르(19.0% → 68.4%),
보스니아(25.0% → 68.1%)
